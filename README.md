# FreeTheBird

**FreeTheBird** is a trivia game built with Next.js and TypeScript where players answer questions to progressively free a caged bird.

[Play Now](https://cagedbird.vercel.app)

---

## Getting Started

```bash
# 1. Clone
git clone https://github.com/damilareoo/freethebird.git
cd freethebird

# 2. Install dependencies
pnpm install

# 3. Run the dev server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

| Tool | Purpose |
|------|---------|
| Next.js | App framework & routing |
| TypeScript | Type safety |
| Tailwind CSS | Styling |
| Vercel | Deployment |
