# FreeTheBird

A trivia game where you answer questions to free a virtual bird from its cage.

🎮 **[Play Now → cagedbird.vercel.app](https://cagedbird.vercel.app)**

---

## How to Play

1. Answer trivia questions correctly
2. Each correct answer brings the bird closer to freedom
3. Get enough right — free the bird!

## Tech Stack

| Tool | Purpose |
|------|---------|
| Next.js | React framework |
| TypeScript | Type safety |
| Tailwind CSS | Styling |
| Vercel | Deployment |

## Getting Started

```bash
# 1. Clone
git clone https://github.com/damilareoo/freethebird.git
cd freethebird

# 2. Install dependencies
pnpm install

# 3. Run locally
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
pnpm build
pnpm start
```

## Project Structure
